# 1 Pager — AI 브랜드 전략 파트너

## 한 줄 요약

브랜딩 디자이너가 기획자 없이도 클라이언트 브랜드 전략을 빠르게 수립할 수 있도록 돕는 AI 웹 도구

---

## 문제 (Problem)

브랜딩 디자이너는 디자인 실력은 있지만, 클라이언트 프로젝트에서 전략 수립까지 요구받는 경우가 많다.

프리랜서나 소규모 스튜디오에는 기획자가 없어 포지셔닝 설정, 타겟 페르소나 정의, 브랜드 스토리 작성 등을 혼자 감당해야 한다.

결과적으로 전략 문서 작성에 과도한 시간이 소요되고, 완성도도 들쑥날쑥하다.

---

## 솔루션 (Solution)

클라이언트 기본 정보와 레퍼런스 이미지를 입력하면 AI가 브랜드 전략 핵심 요소를 자동 도출하고, 탭 형태로 결과를 보여주는 웹 앱을 만든다.

최종 결과물은 PPT 파일로 다운로드해 디자이너가 직접 수정 후 클라이언트에게 공유한다.

**사용 흐름**

```
① 웹사이트 접속
② 클라이언트 정보 + 레퍼런스 이미지 입력
③ "브랜드 전략 설계하기" 버튼 클릭
④ AI 분석 → 탭 형태로 전략 문서 확인
⑤ .pptx 파일로 다운로드
⑥ 직접 검토 및 수정 후 클라이언트 공유
```

---

## 타겟 사용자 (Target User)

- 기획자 없이 혼자 일하는 프리랜서 브랜딩 디자이너
- 소규모 브랜딩 스튜디오 디자이너
- 브랜드 전략을 처음 접하는 주니어 디자이너

---

## 왜 만드는가 (Why)

나 자신이 이 문제를 직접 겪고 있기 때문이다.

전략 문서를 혼자 작성할 때마다 막히고 시간이 낭비된다. 같은 상황의 브랜딩 디자이너들이 전략 작업 시간을 줄이고, 더 완성도 높은 결과물을 낼 수 있도록 돕고 싶다.

---

## 기대 효과

- 전략 문서 작성 시간 단축 (수 시간 → 수십 분)
- 탭으로 섹션별 확인 후 PPT로 수정 및 클라이언트 공유 가능
- 디자이너가 디자인 본업에 더 집중할 수 있는 환경 확보
