# PRD — AI 브랜드 전략 파트너

## 1. 서비스 개요

| 항목 | 내용 |
|---|---|
| 서비스명 | AI 브랜드 전략 파트너 |
| 목적 | 브랜딩 디자이너가 기획자 없이 브랜드 전략 문서를 빠르게 작성할 수 있도록 지원 |
| 형태 | 웹 앱 |
| 주요 사용자 | 프리랜서 브랜딩 디자이너, 소규모 스튜디오 디자이너 |

---

## 2. 핵심 사용자 흐름 (User Flow)

```
① 웹사이트 접속
② 클라이언트 정보 입력 폼 작성
③ "브랜드 전략 설계하기" 버튼 클릭
④ AI 분석 → 탭 형태로 결과 확인
⑤ .pptx 파일로 다운로드
⑥ 디자이너가 직접 검토 및 수정 후 클라이언트 공유
```

---

## 3. 기능 요구사항

### F1. 정보 입력 폼

| 입력 항목 | 설명 | 필수 여부 |
|---|---|---|
| 브랜드/클라이언트명 | 프로젝트 이름 | 필수 |
| 업종 | 패션 / 식음료 / 뷰티 등 | 필수 |
| 타겟 지역 | 국내 / 해외 / 특정 지역 입력 (시장조사 기준) | 필수 |
| 경쟁 브랜드 | 참고할 또는 피해야 할 브랜드 | 선택 |
| 원하는 느낌 | 키워드 자유 입력 (예: 따뜻한, 고급스러운) | 필수 |
| 주요 타겟 | 연령대, 성별, 직업 등 | 필수 |
| 레퍼런스 이미지 첨부 | 무드보드, 경쟁사 로고, 방향성 참고 이미지 등 | 선택 |
| 추가 메모 | 특이사항, 클라이언트 요청 등 | 선택 |

---

### F2. AI 전략 생성

- 입력값 + 첨부 이미지를 함께 분석하여 Claude API 호출
- 이미지가 있을 경우 무드, 컬러 계열, 스타일 방향 등을 추가 반영
- Case Study는 웹 검색 기반으로 실시간 수집
- 아래 7개 섹션 결과 자동 생성

---

### F3. 결과 화면 (탭 형태)

| 탭 | 내용 |
|---|---|
| 브랜드 포지셔닝 | 시장 내 위치, 경쟁사 대비 차별점 |
| 타겟 페르소나 | 주요 고객 프로필, 라이프스타일, 니즈 |
| 브랜드 퍼스낼리티 & 톤앤매너 | 브랜드 성격, 언어 톤, 감성 방향 |
| 브랜드 스토리 | 브랜드가 존재하는 이유, 핵심 메시지 초안 |
| 슬로건 & 카피 | 슬로건 후보 3~5개 + 각 카피 방향성 설명 |
| 디자인 방향성 키워드 | 컬러 계열, 폰트 무드, 이미지 스타일 키워드 |
| Case Study | 업종 + 지역 기준 국내/해외 유사 브랜딩 우수 사례 실시간 검색 |

- 상단에 7개 탭 고정 노출
- 탭 클릭 시 해당 섹션 내용 표시
- 각 탭 내 텍스트는 읽기 전용으로 표시

---

### F4. PPT 다운로드

- "PPT로 다운로드" 버튼 클릭 시 .pptx 파일 생성
- 슬라이드 구성: 탭 1개 = 슬라이드 1장 (총 7장)
- 슬라이드 디자인: 미니멀 & 모던 스타일
  - 흰 배경, 여백 넉넉하게
  - 섹션 타이틀 강조 컬러 1가지
  - 불필요한 장식 없이 텍스트 중심 레이아웃
- 디자이너가 PowerPoint / Keynote에서 바로 수정 가능한 형태

---

## 4. 비기능 요구사항

| 항목 | 내용 |
|---|---|
| AI 응답 시간 | 생성 버튼 클릭 후 60초 이내 결과 표시 (웹 검색 포함) |
| 로그인/회원가입 | 불필요 |
| 데이터 저장 | 저장 기능 없음, 세션 내 사용 후 종료 |
| 지원 브라우저 | Chrome 기준 |

---

## 5. 제외 범위 (Out of Scope)

- 결과 저장 및 히스토리 관리
- 클라이언트에게 직접 공유하는 기능
- 디자인 시안 생성
- 로그인 / 회원 관리
